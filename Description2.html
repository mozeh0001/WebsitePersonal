Smart Robot For Children

OpenCV | Mediapipe | Initial State | C++ | Particle IoT | DeepFace

An interactive robot designed for children that mimics human-like movements, displays facial expressions, and promotes physical engagement.





Robot in Action

Developer Documentation



Overview

  • Uses Particle Argon and servos to mimic human movements
  • Displays facial expressions via MicroOLED screen
  • Uses DeepFace, OpenCV, and MediaPipe for emotion and body detection
  • Encourages active interaction over passive screen time
  • Logs user interactions and emotions for insights


Heart Rate Monitoring for Child Stress Levels

  • Monitors real-time BPM to detect stress
  • Data is sent via Particle Cloud API
  • Heart frame animation for user-friendly displayablity

    • Interactive Heart State - BPM Reading



Facial Recognition & Emotion Display

  • Detects human faces using OpenCV
  • Analyzes emotions using DeepFace AI
  • Displays real-time graphical emotions on the OLED screen

Emotions Displayed on OLED



Body Mimicking & Motion Tracking

  • Uses MediaPipe to track and mimic human movements
  • Robot can wave, kick, and replicate poses
  • Real-time servo angle calculations adjust limb positioning

Body Tracking - MediaPipe Diagram



Remote Control & Blynk App

  • Controlled via the Blynk app
  • Sliders allow manual control of joints for real-time pose adjustments
  • Facial expression tracker updates the OLED screen

Blynk Interface - Control Robot Remotely with App



Data Tracking & Analytics

  • Tracks how often each emotion is displayed
  • Stores interaction patterns and trends
  • Provides behavioral insights over time

Child Mood Over Time



Hardware Components

  • 10 servos for joint control (arms, legs, elbows, hips)
  • MicroOLED display for facial expressions
  • PIR sensor detects human presence
  • Heart rate sensor for real-time BPM tracking
