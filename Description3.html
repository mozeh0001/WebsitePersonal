LED Hand Reactivity

OpenCV | Mediapipe | C++ | Particle IoT

A real-time hand tracking system using MediaPipe, OpenCV, and Particle Argon to detect finger positions and control LEDs accordingly.

  • Captures video frames and processes them in RGB
  • Detects hand landmarks and determines finger positions
  • Maps finger states to corresponding LED colors
  • Sends finger data to the Particle device for updates
  • Displays the raised finger count on the video feed
  • Registers the `setLEDs` function for remote LED control
  • Works with both left and right hands
  • Configures device ID, access token, and API for communication for real-time updates
  • it uses the particle cloud to remote LED gestures
