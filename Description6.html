QuestMates

iOS Swift | Firebase | WeatherStack API

An application inspired by Pokemon Go, Pikmin Bloom and ROBLOX, QuestMates is designed for IOS for users to join and match with each other to compete in AR games to gain currency and personalize their experience by customizing in-game avatars or "go for a walk" to communicate with other users via Firebase. They can join chat rooms while the app's background updates based on weatherstack API upon asking for the user's location.

The App in Action!

  • Coins and Avatar data saves to disk upon closing and opening app.
  • Customizable avatar
  • Game matching to complete in an AR space via game center.
  • Real-time weather background from WeatherStack API and user's specific location.
  • "Take a walk" to talk with friends via chat rooms using Firebase firestore database.

Apple Frameworks

  • GameKit to access game center and compete in multiplayer location-based challenges in real-time with matchmaking via user authentication.
  • SceneKit for avatar customization and "going for a walk," handles the 3D rendering and animations for characters and environments.
  • ARKit Scene tracking to dfor users to compete in a one minute challenge.
  • MapKit to gain access to user's location for weather background updates.
  • SwiftUI: UI layout and user interaction framework for stacking views injecting UIKit into a view as well like joystick and chat box and buttons alerts
  • CoreLocation: Retrieves and manages the user’s geographic location for WeatherStack API background.
  • QuartzCore: Provides animation utilities like CAAnimation.
  • FirebaseFirestore: Handles real-time database operations for chat functionality and chat party rooms.

User Demographics

  • Age Range: 13–45+ years old
