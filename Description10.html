Bridge Breakout Mobile Game

Personal Project | Unity | C# | Blender



Bridge Breakout is an action-adventure game of 5 levels each 2 minutes long where you use a grappling hook to travel across highways and stop bandits from attacking bridges. Your goal is to protect the roads, prevent cars from falling, and take down enemies trying to cause destruction. Swing between broken sections, fight off threats, and keep the highways safe!

Bridge Breakout Gameplay

I created this mobile game in Unity. I planned and designed the levels in Blender, and all assets and scripts are original, including the skybox, except for the modeled bandits.



  • UI of bridge health goes down if cars fall into water or bridge columns are attacked. Left joystick controls movement and right joystick controls camera. Crosshair turns red if enemy in sight and attack button works as followed. Tap on screen to use grapple hook.
  • Line renderer animates for the grappling hook using sin.

    • Controls for Mobile Input (Joystick + Buttons)



  • Used VFX graph to create water splashes when cars fall with physics-based collisions and particle system to make fire + explosions. Body of water is also using shader graph.

    • Water Splash VFX



  • Created the highways, buildings using blender

    • Inspired Level Design. I modeled curved roads in blender and freeway signs.



  • Made the enemy state machine that relies heavily on the navmesh and can jump across bridges and highways using Navmesh links animations sync with jumps.
  • Created an enemy state machine to transition between attacking player, road segements, and bridge columns and patroling.

    • Enemy State Machine

    NavMesh Layout



  • I created a waypoint system for cars to traverse realistically while using object pooling to turn them inactive for good memory usage in Unity's garbadge collector.
